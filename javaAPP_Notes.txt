MENU :

THEY ARE CONTAINED WITHIN A MENUBAR CLICKING ON A NAME OPENS THE MENU CONTAINING MENU ITEMS AND SUBMENU'S. THE CLASS USED TO CREATE A MENU IS JMenu. 
the constructor for creating menu is : 
public Jmenu(String s)
public Jmenu(Action a)
public Jmenu(String s, boolean b)
constructs  a menu with the label s. To add a menu to the menubar the method of the class JMenubar user is add 
protoype : 

public JMenu add( JMenu menu)
 appends the specified menu to the end of the menubar
